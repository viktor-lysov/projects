### Дашборд с основными показателями отдела продаж дистрибьютора медицинского оборудования

##

### Описание проекта

В рамках проекта разработан аналитический дашборд для компании, специализирующейся на продаже медицинских пробирок (для анализа крови, ПЦР и других исследований).

Дашборд предназначен для мониторинга ключевых показателей отдела продаж и поддержки оперативной работы менеджеров по продажам и взаимодействию с поставщиками.

Решение агрегирует данные о продажах, клиентах, поставщиках и регионах, предоставляя удобный интерфейс для анализа и принятия решений.

### Цель проекта

Разработать интерактивный дашборд, который:

* Отображает ключевые метрики продаж на верхнем уровне
* Позволяет анализировать динамику показателей во времени
* Даёт возможность выявлять изменения по поставщикам, клиентам и каналам продаж
* Поддерживает оперативную работу менеджеров через фильтры и детализацию данных

### Бизнес-требования

В ходе проекта были собраны и уточнены требования заказчика.

#### Лист 1 — Обзор для руководителя:

* Ключевые метрики:
  * Общая сумма продаж
  * Количество проданных товаров
  * Средний чек
* Динамика показателей во времени
* Рейтинг товаров (по выручке / количеству)
* География продаж (распределение по регионам)

#### Лист 2 — Рабочий инструмент для менеджеров:

* Анализ продаж по периодам времени
* Мониторинг изменений:
  * по поставщикам
  * по типам клиентов
  * по каналам продаж
* Фильтрация данных:
  * по датам
  * по регионам
  * по поставщикам

### Данные

В проекте используются данные о продажах со следующими ключевыми полями:

* Дата продажи (`date_of_sale`)
* Продукт (`product_id`, `product_name`, `category`)
* Объём продаж (`quantity`)
* Цена и выручка (`sale_price`, `total_sale_amount`)
* География (`region_name`, `population`)
* Поставщики (`supplier_name`)
* Типы клиентов (`customer_type`)
* Каналы продаж (`sales_channel`)

### Функциональность дашборда
* Интерактивные фильтры по ключевым параметрам
* Отображение KPI
* Визуализация динамики продаж
* Анализ структуры продаж (по товарам, регионам, каналам)
* Гео-анализ продаж
* Возможность детализации показателей
